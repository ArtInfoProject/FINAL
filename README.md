# ArtSoda🎨
대충 로고 여기
## 프로젝트 소개
기존 시각예술인들이 제일 많이 사용하는 사이트인 “네오룩" 에서 아이디어를 얻어 ‘개선과 가독성'을 집중주제로 소통을 촉진하는 미술 플랫폼.

## Usage

```python
import foobar

# returns 'words'
foobar.pluralize('word')

# returns 'geese'
foobar.pluralize('goose')

# returns 'phenomenon'
foobar.singularize('phenomena')
```
