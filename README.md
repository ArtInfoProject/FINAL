
# ArtSoda🎨

## 프로젝트 소개
기존 시각예술인들이 제일 많이 사용하는 사이트인 “네오룩" 에서 아이디어를 얻어 ‘개선과 가독성'을 집중주제로 소통을 촉진하는 미술 웹 프로젝트 입니다.
## ⚙개발 환경 & 사용 언어
### 프론트
- Visual Studio, Photoshop, Figma
- HTML, JS, CSS, JSP
### 백엔드
- sts3, Spring
- Java, Tiles, mybatis
### 데이터
- Pycharm, MySQL
- Python, SQL

## ⏳개발 기간
- 1차 개발: 2023.8.1~2023.8.30
- 2차 추가 개발: 2023.10.10~2023.10.25
